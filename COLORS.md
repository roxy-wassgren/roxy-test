# Project Color Palette

This document outlines the color palette used throughout the project, including their semantic meanings and hexadecimal values.

## Color Definitions

### Primary Background & Surface Colors
*   **Name:** `Background Default`
    *   **Hex:** `#f5efe3`
    *   **Description:** The main background color for the page.
    *   **CSS Variable:** `--bg`

*   **Name:** `Panel Background`
    *   **Hex:** `rgba(255, 251, 245, 0.9)`
    *   **Description:** Background color for UI panels, cards, and elevated surfaces, with transparency.
    *   **CSS Variable:** `--panel`

*   **Name:** `Light Cream`
    *   **Hex:** `#f9f2e7`
    *   **Description:** A very light cream color used in gradients for the body background.

### Text & Typography Colors
*   **Name:** `Primary Text`
    *   **Hex:** `#1d231f`
    *   **Description:** The default color for primary text content.
    *   **CSS Variable:** `--text`

*   **Name:** `Muted Text`
    *   **Hex:** `#57625b`
    *   **Description:** Used for secondary or less emphasized text, like descriptions or captions.
    *   **CSS Variable:** `--muted`

*   **Name:** `Header Text`
    *   **Hex:** `#ffffff`
    *   **Description:** White text used within the site header for contrast.

*   **Name:** `Rank Text`
    *   **Hex:** `#fff8f0`
    *   **Description:** Off-white color for text within the rank badge.

### Accent & Interactive Colors
*   **Name:** `Primary Accent`
    *   **Hex:** `#b65a2d`
    *   **Description:** A primary accent color used for key interactive elements, call-to-actions, and decorative elements like the eyebrow text.
    *   **CSS Variable:** `--accent`

*   **Name:** `Secondary Accent`
    *   **Hex:** `#e0b166`
    *   **Description:** A softer accent color, often paired with the primary accent in gradients or highlights.
    *   **CSS Variable:** `--accent-soft`

*   **Name:** `Site Header Background`
    *   **Hex:** `#3d8b5c`
    *   **Description:** The green background color for the main site header.

*   **Name:** `Gradient Soft Yellow/Orange`
    *   **Hex:** `rgba(224, 177, 102, 0.35)`
    *   **Description:** Used in the body's top-left radial gradient.

*   **Name:** `Gradient Reddish-Brown`
    *   **Hex:** `rgba(182, 90, 45, 0.22)`
    *   **Description:** Used in the body's right radial gradient.

### Other
*   **Name:** `Image Placeholder Background`
    *   **Hex:** `#d8d1c3`
    *   **Description:** A neutral background color for image placeholders within cards.
